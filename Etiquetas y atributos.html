Etiquetas para estructurar:

Para este tipo de etiquetas como lo son nav, section, etc. No existen atributos especiales al igual que para las etiquetas de semántica. Sólo existen atributos globales los atributos globales son los siguientes, estos atributos son globales ya que como su nombre lo inidca, se utilizan en distintas etiquetas de diferentes categorías

  1. title
  2. id
  3. class
  4. style
  5. lang
  6. dir

Etiquetas para agrupación:

Para la etiqueta ol tenemos los siguientes atributos:

reversed: con este atributo podemos modificar el comportamiento estándar de una lista ordenada haciendo que se vea listada en orden descendente en lugar de ascendente

start: Define el número inicial de la lista ordenada (si es diferente a 1).

Ejemplo de atributo reversed

  1. Sebas
  2. Carlos
  3. Merari

Ejemplo de atributo start

  1. Sebas
  2. Carlos
  3. Merari

Para la etiqueta ul tenemos los siguientes atributos:

type: Indica el estilo de los items de la lista
compact: Indica que la lista debe mostrarse compacta

Etiquetas para incrustar contenido:

Para la etiqueta img tenemos los siguientes atributos:

alig: Especifica el alineamiento horizontal del elemento

alt: Texto alternativo que se mostrará en el navegador en caso de que la imagen no se pueda mostrar

border: Permite definir el borde de un elemento HTML

height: Permite especificar la altura de los elementos HTML indicados

src: La URL del contenido integrable en la etiqueta HTML que lo acompaña

srcset: Define diferentes versiones de la imagen para mostrar una u otra según el tipo de pantalla

usemap: Indica la URL parcial de un image map asociado con el elemento

width: Para los elementos enumerados aquí, establece el ancho del elemento

Para la etiqueta iframe tenemos los siguientes atributos:

src: La URL de la página a insertar

width: Indica el ancho del marco donde se va mostrar la página web externa

height: Indica la altura del marco donde se va a mostrar la página web externa

allowfullscreen: Atributo tipo booleano. Activa de ser posible, el modo pantalla completa

id: identificador de iframe para usarse en código de JavaScript

name: define el nombre objetivo para el iframe con el fin de ser fácilmente reconocbile, usado desde JavaScript

scrolling: permite la inserción de barras de desplazamiento dentro del iframe. Se puede definir de 3 formas: Yes/no/Auto
